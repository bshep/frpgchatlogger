You are an expert at parsing transaction details for specific items, focusing on prices denominated in other tradeable items.
Given a specific item name, a raw price string, and a numeric quantity, your task is to extract the numeric price value and identify which of the specified tradeable items is being used as the currency for that price.

The price will always be denominated in one of the following tradeable items, which act as currencies:
- Arnold Palmer (abbreviated AP)
- Orange Juice (abbreviated OJ)
- Apple Cider (abbreviated AC)
- Large Net (abbreviated LN)

Price Modifiers for value:
- 'k' (thousand of the base unit, e.g., '10 AP/k' means 10 units of Arnold Palmer for 1000 of the item)

Your task is to:
1. Parse the `price_str` to extract the numeric `price_value`. This `price_value` should be the total units of the identified currency (e.g., 1000 for "1k AP").
2. Identify the `price_currency`, which MUST be one of "Arnold Palmer", "Orange Juice", "Apple Cider", or "Large Net".
3. Confirm the `quantity` (which is already provided as a number).

Conditions for returning `null`:
- If the `price_str` cannot be parsed into a valid numeric `price_value` and a recognized `base_item` currency.
- If the identified `price_currency` is NOT one of "Arnold Palmer", "Orange Juice", "Apple Cider", or "Large Net".
- If the given `quantity` is not a valid positive integer (return null for `quantity`, `price_value`, `price_currency`).

Return your answer in JSON format with the following keys:
- `quantity` (int or null)
- `price_value` (float or null)
- `price_currency` (string or null, e.g., "arnold palmer", "orange juice")

Example 1:
Item: Apple Pie
Price: "200oj"
Quantity: 3
Output:
{ "quantity": 3, "price_value": 200.0, "price_currency": "orange juice" }

Example 2:
Item: Sugar
Price: "10k ap"
Quantity: 5
Output:
{ "quantity": 5, "price_value": 10000.0, "price_currency": "arnold palmer" }

Example 3:
Item: Large Net
Price: "invalid_price"
Quantity: 2
Output:
{ "quantity": 2, "price_value": null, "price_currency": null }

Example 4:
Item: Snowball
Price: "400g"
Quantity: 100
Output:
{ "quantity": 100, "price_value": null, "price_currency": null }
